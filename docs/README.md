# Documentation du dépôt Henrion-themes

Ce dossier regroupe les conventions IA et les contrats officiels du système de thèmes Henrion.

- `ai/AI-CONVENTIONS.md` : conventions transverses.
- `ai/AI-THEMES.md` : architecture des packages et manifestes de thèmes.
- `ai/AI-MARKETPLACE.md` : format et règles du catalogue marketplace.
- `contracts/THEME-CONTRACT.md` : contrat normatif du `theme.json`.
- `contracts/MARKETPLACE-CONTRACT.md` : contrat normatif du `public/index.json`.
