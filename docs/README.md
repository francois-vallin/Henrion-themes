# Documentation du dépôt Henrion-themes

Ce dossier regroupe les conventions IA et les contrats de compatibilité utilisés pour maintenir un catalogue de thèmes Henrion cohérent.

- `ai/` : conventions de structuration et d'implémentation.
- `contracts/` : contrats officiels des thèmes et du catalogue marketplace.

Référentiel actuel : logique v0.4 (compatibilité minimale Henrion `0.3.3`).
Toute évolution des thèmes publiés dans `packages/` doit rester compatible avec ces documents.
